# Словарь данных

## Таблица players_profile

Содержит информацию о зарегистрированных игроках и их основных характеристиках.

| Поле              | Описание                               |
| ----------------- | -------------------------------------- |
| player_id         | Уникальный идентификатор игрока        |
| registration_date | Дата регистрации игрока                |
| platform_type     | Платформа устройства (iOS или Android) |

---

## Таблица player_activity_log

Содержит информацию об игровых сессиях пользователей.

| Поле               | Описание                               |
| ------------------ | -------------------------------------- |
| player_id          | Уникальный идентификатор игрока        |
| session_start_time | Дата и время начала игровой сессии     |
| session_end_time   | Дата и время завершения игровой сессии |

---

## Таблица purchase_events

Содержит информацию о внутриигровых покупках.

| Поле              | Описание                          |
| ----------------- | --------------------------------- |
| player_id         | Идентификатор игрока              |
| purchase_datetime | Дата и время покупки              |
| asset_id          | Идентификатор игрового объекта    |
| purchase_quantity | Количество приобретённых объектов |

---

## Таблица invite_network

Содержит данные реферальной программы.

| Поле            | Описание                                       |
| --------------- | ---------------------------------------------- |
| player_id       | Идентификатор игрока, отправившего приглашение |
| friend_reg_flag | Признак регистрации приглашённого пользователя |

---

## Таблица item_price_history

История изменения стоимости игровых объектов.

| Поле               | Описание                       |
| ------------------ | ------------------------------ |
| asset_id           | Идентификатор игрового объекта |
| asset_price        | Стоимость объекта              |
| price_period_start | Дата начала действия цены      |
| price_period_end   | Дата окончания действия цены   |

---

## Таблица game_assets_catalog

Справочник игровых объектов.

| Поле           | Описание                       |
| -------------- | ------------------------------ |
| asset_id       | Идентификатор игрового объекта |
| asset_name     | Название игрового объекта      |
| asset_category | Категория игрового объекта     |
