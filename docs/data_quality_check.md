# Проверка качества данных

## Проверка таблицы players_profile

Для оценки качества данных была проведена проверка идентификаторов игроков.

### Результаты проверки

| Проверка                  | Результат |
| ------------------------- | --------- |
| COUNT(*)                  | 3078      |
| COUNT(player_id)          | 3078      |
| COUNT(DISTINCT player_id) | 3078      |

### Вывод

Проверка показала, что таблица `players_profile`:

* не содержит NULL-значений в поле `player_id`;
* не содержит пустых значений;
* не содержит дубликатов по `player_id`.

Каждая строка таблицы соответствует одному уникальному игроку, поэтому данные могут использоваться для дальнейшего анализа без дополнительной очистки.
