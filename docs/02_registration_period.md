# Исследование периода регистрации пользователей

## Бизнес-вопрос

Необходимо определить временной период формирования пользовательской базы игры:

- когда зарегистрировался первый пользователь;
- когда зарегистрировался последний пользователь;
- есть ли пользователи с отсутствующей датой регистрации.

Проверка позволяет оценить полноту данных о пользователях и определить временные рамки дальнейшего продуктового анализа.

---

## Источник данных

Таблица:

`planet_hunt.players_profile`

Используемое поле:

`registration_date` — дата регистрации пользователя.

---

## Логика анализа

Для проверки были рассчитаны следующие показатели:

| Метрика | Описание |
|---|---|
| NULL registration date | количество пользователей без даты регистрации |
| MIN(registration_date) | первая регистрация в системе |
| MAX(registration_date) | последняя регистрация в системе |
| COUNT(registration_date) | количество пользователей с заполненной датой регистрации |

---

## Результат анализа

По результатам расчёта:

- первая регистрация пользователя: **01.07.2022**
- последняя регистрация пользователя: **30.03.2023**
- количество пользователей без даты регистрации: **0**

---

## Вывод

Данные о регистрации пользователей заполнены полностью.

Пользовательская база игры сформирована за период:

**июль 2022 — март 2023**

Отсутствие пропусков в поле `registration_date` позволяет использовать дату регистрации для дальнейшего анализа:

- когорт пользователей;
- retention-метрик;
- динамики привлечения игроков;
- эффективности маркетинговых активностей.
