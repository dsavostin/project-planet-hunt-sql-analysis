# Project: Planet Hunt v.0.3.1

Анализ продуктовых метрик мобильной игры

## Бизнес-задача

Команда игры хочет понять:
- насколько активно пользователи взаимодействуют с продуктом;
- какие маркетинговые активности дают рост;
- какие факторы влияют на монетизацию.

## Используемые данные

- пользователи
- игровые сессии
- покупки
- реферальная программа
- каталог игровых объектов

## Что сделано

✓ проверка качества данных
✓ анализ активности пользователей
✓ анализ маркетинговой акции
✓ когортный анализ
✓ анализ монетизации
✓ поиск лояльной аудитории

## Основные результаты

- акция марта увеличила активность пользователей;
- новые когорты показывают более высокую ценность;
- реферальная программа является источником дополнительного роста.

## Инструменты
